# Event App Template System

This project is now structured to be reused for ANY event.

## 🔁 How to reuse for a new event

1. Copy the repo
2. Update ONE file:

👉 `frontend/src/data/eventConfig.ts`

---

## 🧩 What to change

### Event Info
- Name
- Dates
- Location
- Tagline

### Home Actions
- Tickets URL
- Merch URL

### Sponsors
👉 `frontend/src/data/sponsors.ts`

- Add real businesses
- Add URLs
- Add tiers

---
